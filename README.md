# JSON Tools

## Description

A CLI application that converts json file to yaml file. It can also verify fields and update fields.

## Features

* Verify if fields are correct
* Update a desired field
* Convert the json file to yaml file

## How to run 

1. Clone the repo
2. run the command `go build main.go`, you will get binary file
3. run the app with `--input` and `--output` flags: `./main --input <input file> --output <output file>`
